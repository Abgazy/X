# XSaver - Twitter (X) Video Downloader

أداة بسيطة وسريعة لتحميل فيديوهات منصة إكس (تويتر سابقاً) بجودة عالية وبدون علامة مائية.

## المميزات | Features
- تحميل فيديوهات X بجودة HD.
- واجهة مستخدم متوافقة مع الجوال والكمبيوتر.
- حماية وتشفير للطلبات عبر Cloudflare Workers.
- مجاني بالكامل وسريع الاستجابة.

## رابط الموقع | Live Demo
[https://abgazy.github.io/X/](https://abgazy.github.io/X/)

## التقنيات المستخدمة | Tech Stack
- HTML5 & CSS3 (Twitter UI Theme)
- JavaScript (Vanilla)
- Cloudflare Workers (Backend Proxy)
- RapidAPI (X Downloader API)
